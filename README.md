# Blinkit Sales Analysis Dashboard | Power BI

## Project Overview
This project presents an interactive Power BI dashboard for analyzing Blinkit's sales data. The dashboard provides meaningful insights into sales performance, product categories, outlet performance, customer ratings, and business trends.

## Key Insights Covered
- Total Sales Performance Analysis
- Item Category Analysis
- Outlet-wise Sales Comparison
- Outlet Size and Location Analysis
- Customer Rating Analysis
- Fat Content Analysis
- Product Performance Trends

## Tools & Technologies
- Power BI
- DAX
- Data Visualization
- Data Analytics

## Dashboard Features
- Interactive filters and slicers
- KPI cards for business metrics
- Trend analysis visuals
- Category and outlet-level insights
- User-friendly dashboard layout

## Objective
To transform raw sales data into actionable business insights that help understand customer behavior, sales patterns, and operational performance.
