# 🛒 BlinkIT Sales & Outlet Performance Dashboard | Power BI

## 📌 Project Overview

This project showcases an interactive **BlinkIT Sales & Outlet Performance Dashboard** built using **Power BI**. The dashboard provides valuable insights into sales performance, outlet distribution, customer ratings, and product categories through interactive visualizations and KPI tracking.

---

## 📸 Dashboard Preview

![BlinkIT Sales & Outlet Performance Dashboard](BlinkIT%20Sales%20%26%20Outlet%20Performance.png)

---

## 🎯 Objectives

- Analyze overall sales performance
- Compare outlet performance across different locations and sizes
- Identify top-performing product categories
- Track customer ratings and average sales
- Monitor outlet establishment trends
- Enable data-driven business decisions

---

## 📊 Key Performance Indicators (KPIs)

| KPI | Description |
|------|-------------|
| 💰 Total Sales | Overall revenue generated |
| 📦 Total Items | Number of items sold |
| ⭐ Average Rating | Average customer rating |
| 💵 Average Sales | Average sales per item |

---

## 📈 Dashboard Features

- Sales Analysis
- Outlet Performance Analysis
- Item Type Analysis
- Fat Content Analysis
- Outlet Size Distribution
- Outlet Location Analysis
- Outlet Type Comparison
- Outlet Establishment Trend
